# api_gemini Decision Registry

Tracks open questions and architectural decisions that tasks may close. Each entry has a Q-NN identifier referenced in task `**Closes:**` fields.

### Responsibility Table

| Column | Purpose |
|--------|---------|
| `Q-NN` | Unique decision identifier (permanent, never retired) |
| `Question` | The open question or decision to be made |
| `Status` | Open / Closed |
| `Closed By` | Task ID that resolved this question |
